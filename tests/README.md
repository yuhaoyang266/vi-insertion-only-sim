# Test Layout

- `artifacts/`: manifest, provenance, and artifact contract checks.
- `core/`: lightweight import and package-boundary checks.
- `paper/`: paper asset, table, figure, prose, and evidence-matrix synchronization checks.
- `packaging/`: anonymous submission bundle builder tests.
- `reviewer/`: minimal no-training smoke tests copied into anonymous snapshots.
- `runners/`: CLI and experiment runner entrypoint tests.
- `sprints/`: sprint-level documentation and output contract checks.
- `three_dof/`: environment, policy, benchmark, support metric, and training-contract unit tests.
