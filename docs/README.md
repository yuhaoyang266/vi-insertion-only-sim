# Documentation Layout

- `figure_asset_manifest.md`: reviewer-facing figure and artifact source manifest. Kept at `docs/` root because the anonymous snapshot copies this exact file.
- `plans/`: execution plans and milestone checklists.
- `project/`: long-running project state, including task plan, progress log, findings, narrative choices, and protocol freeze notes.
- `submission/`: editor-facing submission notes, cover-letter draft, and upload checklist material.
- `reviews/`: internal or external review reports.
