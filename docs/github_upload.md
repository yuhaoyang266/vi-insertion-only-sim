# GitHub Upload Instructions

The package is prepared for this target URL:

`https://github.com/yuhaoyang266/vi-insertion-only-sim-paper`

This machine does not currently have the GitHub CLI (`gh`) installed, and the source repository's
`origin` remote points to a local filesystem path rather than GitHub. To publish this package,
create the GitHub repository first, then run the commands below from this directory.

```bash
git init
git branch -M main
git add .
git commit -m "docs: add paper project package"
git remote add origin https://github.com/yuhaoyang266/vi-insertion-only-sim-paper.git
git push -u origin main
```

After pushing, the URL embedded in `paper/main.tex` will resolve to the public project page.

## Suggested Cover Letter Positioning

This manuscript is best introduced as a benchmark-local, teacher-coupled learnability study rather
than as a teacher-independent causal theorem:

> The submitted package studies a controlled 3DoF analytical insertion benchmark in which BC
> demonstrations are generated by a variable-impedance teacher. The resulting claim is therefore a
> benchmark-local, teacher-coupled learnability diagnosis, not a general theorem about all
> demonstration designs or a broad algorithm ranking.
